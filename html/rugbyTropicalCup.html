Rugby Tropical Cup

Duration : from 01/05/23 to ...

Purpose :To allow a company to generate tickets with the teams' name, the stadium's location and the date, then scan their QR Code.

Programming languages :Django, Python, HTML / CSS

Softwares : : GitHub, Open Classroom, VS Code

Presentation :This web application allows to recover informations troughout an API about all the rugby matches. It was made with a website base that contained : 1 page for every stad's location, 1 page for every teams that will participate in the matches, and a page for the newsletter.

Fonctionalities : Consult every teams, stad's location, and informations about the Rugby soprt and Japan that will welcome all the teams. Generate tickets with its ID that will display the match informations. Scan the tickets' QR Code.
