Programming languages / Softwares :
API Kitsu.io, GitHub, Django (version : ), Python (version : ), HTML / CSS, Figma, Open Classroom, VS Code (version : )

Graphic charter

LOGO
For the logo, we made it simple with a generic character's face to remind the anime world in addition to this website'name Nihon Collection.

PROJET'S NAME
Japan is famous worldwide for their anime so we chose the name Nihon Collection, Nihon which means Japan in japanese and collection for all the anime accessible in the API Kitsu.io.

COLORS
As for colors, what we see the most in anime is the floral landscape with cherry flower. So the main color is pink.
